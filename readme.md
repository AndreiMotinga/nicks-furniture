#### Deployment
1. Create bucket
2. Create `.env` file with

```
AWS_ACCESS_KEY_ID=xxxxxxxxxxxxxxxxxxxx
AWS_SECRET_ACCESS_KEY=xxxxxxxxxxxxxxxxxxxx
BUCKET=xxxxxxxxxxxx
```

3. run
```
./deploy.sh
```
